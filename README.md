# 🎓 Student Management System (Python)

## 📌 Project Overview

This project is a Python-based application designed to manage student records efficiently. It allows users to perform operations such as adding, updating, deleting, and viewing student data.

The system demonstrates the use of core Python concepts and basic data management techniques.

---

## 🛠️ Technologies Used

* Python
* File Handling
* Basic Data Structures (Lists, Dictionaries)

---

## ⚙️ Features

* Add new student records
* View all student details
* Update existing student information
* Delete student records
* Search students by ID or name

---

## 🧠 Concepts Used

* Functions and modular programming
* Conditional statements and loops
* File handling (read/write)
* Data structures for storing records

---

## ▶️ How to Run

### 📌 Prerequisites

* Python 3.x installed

### 🚀 Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-management-system-python.git
```

2. Navigate to the folder:

```bash
cd student-management-system-python
```

3. Run the Python file:

```bash
python main.py
```

---

## 📁 Files Included

- `main.py` → Python script containing all functionalities (add, update, delete, view student records)
